项目一：bistumaker.cn（社团官网）

一、使用工具

​ 前端(Vscode)：Vue3 + Pinia + Element-plus + Vue-Router

​ 后端(Intellij)：SpringBoot3 + Mysql + Mybatis

​ 版本管理：git

二、现阶段成果

​ 成果链接：bistumaker.cn

​ 项目链接：Tecalion/BistuMaker: 后续维护 (github.com)

三、项目概述

​ 因后端主要开发人员不是我，此处作品集仅包含我主要负责的前端部分，开发初期我并不熟悉idea内部的git提交方法，故整合的1.0版本由学长提交，后期维护由我fork到自己的仓库中进行。

​ *网页的内容更新可在后台完成，技术更新经由腾讯云服务器的宝塔面板进行。

1.项目概述

​ 本校创客空间社团的信息官网(bistumaker.cn)，前端采用Vue3 + Pinia + Element-plus + Vue-Router进行开发,后端使用SpringBoot3 + Mysql + Mybatis，由两名学生共同开发，社团指导老师提供资金支持购买服务器与域名。

2.人员分工

​ 初期开发由两人共同完成，大二下的我负责前端的前台页面设计，另一位大三下的学长负责后端以及前端的后台页面设计，与学长商讨调研后，决定购用腾讯云的轻量级服务请与域名，由我完成域名备案与使用宝塔面板完成网站挂靠服务器。

​ 初期版本正式运行并投入使用后，由我负责全栈的bug修复与前后台样式优化。

3.后续安排

​ 目前已经完成一次较大的细节和互动体验优化，后续会持续优化维护该网站，进行内容更新或是技术更新。

四、该项目对我的意义

​ 本项目源于我大学期间一直参与和组织的社团。在这个社团中，我学习并了解了许多拓展视野的知识，如Web搭建、鸿蒙开发、机器学习和共享算力等。对于在编程方面从大学才开始起步的我来说，这些知识尤为珍贵。我非常感激这个社团开源开放的精神和合作互助的氛围。

​ 这是我我初次接触接近现代企业标准的项目，项目整体使用Vue和SpringBoot分别主导前后端的开发工作，通过git拉取和提交代码，其中我负责了大部分的前端开发，学习并实践了Vue的相关知识。

​ 在该项目的前端工程中，我在home页面的StarField组件中采用了Vue 2的选项式API风格。在这个组件的设计过程中，我对Vue生态的理解逐渐深入。随后，在其他页面和其他组件中，我采用了Vue 3的组合式API风格，这使得组件逻辑更加灵活和清晰。通过这个项目的完成，我对Vue的理解和掌握有了显著的提升。这次经历不仅让我熟悉了Vue的不同API风格，还增强了我的代码组织能力和项目管理能力。
