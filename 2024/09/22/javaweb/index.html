宿舍管理系统

项目四：宿舍管理系统(Javaweb结课作业)

一、使用工具

​ 前端：HTML5 + CSS3 + JavaScript + bootstrap

​ 后端：Tomcat9 + Mysql +Java

二、现阶段成果

​ 已提交课设，可查看运行截图或运行项目查看效果（数据库自带账户admin 密码111）

三、项目概述

1.项目概述

​ 一个MVC架构的宿舍管理系统，学业内暑假小学期Javaweb的结课作业，老师要求使用Tomcat服务器挂载页面，以及用JSP页面实现前端，数据库自选。

2.人员分工

​ 单人开发。

四、该项目对我的意义

​ 主要收获是在本次项目内实践了bootstrap框架的使用，加深了我对前端ui框架工具的认识，激发了我对了解学习更多前端ui框架的热情。我希望能有更多实践机会和实际的开发需求，以此磨练并提高我对前端UI的设计能力。
