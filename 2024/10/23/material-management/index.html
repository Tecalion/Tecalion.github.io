智能物料管理

2024/10/23

校园智能物料管理系统

一、使用工具

​ 前端(Vscode)：Vite + Vue3 + Vue Router + Pinia + Element-Plus + Node.js

​ 后端(Intellij)：SpringBoot3 + Sa-Token + JDBC + Druid + Redis +Spring Security

​ 版本管理：Git

二、现阶段成果

​ 目前仍在开发中，因为是校方私有项目，暂时不便展示。

​ 可查看运行截图确认部分效果。

三、项目概述

​ *该项目库虽然由我创建，但老师说明该项目是校内私有项目，不开源，于是设为私有，如果需要大概浏览可以让我个人展示。

1.项目概述

​ 在读学校的在建项目，预计年底或明年初落地，主要用于协调本校各单位（如项目部、材料部、建筑部等）的工作流程，方便各单位员工查看所需信息与任务。

2.人员分工

​ 指导老师搭建项目大体架构，我与另一名本科大三同学负责不同页面的具体前后端实现，前端vue工作主要是设计页面结构、通过axios连接后端对应api以正确展示列表数据。后端java工作主要是优化sql语句和方法，通过redis缓存发送给前端的列表数据。

3.后续开发

​ 由于原本同组的三位大三同学因项目难度偏高或日常事务较多退组，组内新拉入了两名研究生成员，后续还会持续完善优化，加速开发尽快落地。

四、该项目对我的意义

​ 在这个项目中，我和另一位大三的同学负责与甲方（本校建筑部）沟通需求和跟进开发进度。这是我首次接触代码量最大、形式最正式的项目。尽管遇到了许多新知识，但我们一直在开发过程中不断学习和进步。

​ 前期的主要困难在于不熟悉Vue前端项目与后端SpringBoot项目的分离结构。得益于我在其他项目中的经验，Vue前端界面开发对我来说并不算难事。经过一段时间的学习，我成功掌握了通过Axios库发送HTTP请求与后端进行通信的方法。在后端方面，我也了解了通过Tomcat服务器和SpringBoot的Rest API来接收这些请求。第一个功能模块开发完成后，后续类似模块的开发变得更加顺利。

​ 通过这个项目，我深刻体会到编程设计水平只有在实际开发中才能得到最大的提升。我希望未来能接触更多类似的大型项目，进一步熟练使用Element-Plus等前端UI库，设计出更加美观和实用的界面。
